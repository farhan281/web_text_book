A judge's act is binding judgment, unless he acts as agent or distributor.

If the act is a judicial act and not an exception, then it is binding. The judge cannot reverse it, and it stands even if new facts emerge.

Learning Objective

Core Idea
Jahalah prevents validity in most transactions, but exceptions exist in claims, bequest, agency, divorce, manumission, and hudud.

This lesson presents a comprehensive list of how jahalah (ignorance or ambiguity) affects various legal acts according to the Hanafi school. Guided Path

Only beneficial restrictions or those emphasized with negation bind the agent.

This lesson explores a foundational maxim from Al-Ashbah wa al-Naza'ir that governs how restrictions placed by a principal on an agent are treated in Hanafi law. This lesson examines a key Hanafi legal maxim from al-Ashbah wa al-Naza'ir. The maxim states that whoever owns disposal of a thing also owns its part. Core Idea

This lesson examines a nuanced exception to a general Hanafi maxim on agency. The maxim states that agency is permitted for any act the agent can perform for himself. However, the executor (wasi) of an orphan's estate has a special fiduciary role that creates an exception: he may purchase the orphan's property for his own benefit, but he cannot be appointed as an agent to buy the same property for someone else. Understanding this exception requires careful attention to the capacity in which the executor acts.

Guided Path
1
Recall the general maxim
Start by recalling the general maxim: 'يجوز التوكيل بكل ما يعقده الوكيل لنفسه' – agency is permitted for everything that the agent can contract for himself. This means that any person can appoint an agent to do something that the agent could do on his own behalf.
This is a sughra (minor) maxim from the Hanafi qawā'id corpus.

The passage says 'ومما خرج عن قولهم' (and among the exceptions from their statement). The exception is the executor (الوصي) of an orphan. The reasoning given is 'والنفع ظاهر' (the benefit is apparent) for buying for himself. When the executor buys for himself, the benefit to the orphan is clear (e.g., fair price, need for liquidity). The Hanafi Distinction Between Agents for Debt and Property After Principal's Death

Learning Objective
By the end of this lesson, you will be able to apply the Hanafi distinction between an agent for debt and an agent for specific property to determine whether the agent's claim of receipt is accepted after the principal's death.

Core Idea
An agent for debt is not believed after the principal's death; an agent for property is believed if the property is consumed.

The death of a principal raises the question: can an agent claim to have completed his task? Hanafi fiqh distinguishes based on the type of agency. An agent for debt seeks to discharge the deceased's liability, creating suspicion. An agent for specific property seeks to remove his own liability, so his claim is accepted when the property is consumed. This distinction, found in later fatwa works, is a key tool for muftis.

Guided Path

Determine the Agent's Claim

3
Apply the Hanafi Distinction
For agent for debt: claim is not accepted because he would remove liability from the deceased. For agent for specific property: claim is accepted if the property is consumed (perished), not if still in existence.
Rationale (talil): debts are paid in kind (mithli), so the estate is discharged; for specific property, the agent is liable for loss if not delivered.
4
Consider Exceptions and Opposition
Note that some analogize to the agent for deposit, but al-Walwaliji's distinction prevails. The agent for deposit is not believed because he also seeks to remove liability from the deceased, similar to debt.
This is the thamarah al-khilaf: the practical outcome differs.

Formulate the Fatwa
Based on the mufta bihi position, state the ruling for the given scenario. The agent for debt is not believed; the agent for property is believed if the item is consumed.

Maxim with exceptions and furuat

Learning Objective
By the end of this lesson, you will be able to state the Hanafi maxim on termination of agency upon death, identify the exception of bay' wafa, and apply the furuat to new cases.

Core Idea
Agency ends at the principal's death, except for a sale with repurchase (bay' wafa).

This passage from Al-Ashbah wa al-Nazair presents a legal maxim (qa'ida) governing the effect of death on agency contracts. While the general rule is that the principal's death invalidates the agency, Hanafi fiqh carves out an exception for a specific transaction: bay' wafa (sale with repurchase). Understanding this exception requires both knowledge of the maxim and its application in furuat, as well as the reasoning (istihsan) that supports it. The fatwa authority (Munyat al-Mufti) further refines the rule by excluding currency exchange.

Guided Path
1
State the general maxim
The general rule: 'If the principal dies, the agency becomes void.' Understand that agency is a revocable contract that terminates automatically on death.
This is the base rule from which exceptions arise.
2
Identify the exception
Agency for bay' wafa (sale with repurchase) continues after death. The transaction is incomplete because the seller retains a right to repurchase, so death does not automatically void the agency.
Bay' wafa is a Hanafi concept considered a valid sale.
3
Analyze the istihsan validation
If the principal himself receives the price from the buyer, the agent's prior act is validated by istihsan (juristic preference), even though the agency was technically voided by death.
Istihsan overrides strict analogy to prevent loss.
4
Note the sarf exclusion
The istihsan validation does not apply in currency exchange (sarf) because of the strict requirement of hand-to-hand exchange (taqabud), which cannot be fulfilled after death.
This exception is from Munyat al-Mufti.
5

Distinguish between cases: agency for ordinary sale (void upon death, no validation) vs. bay' wafa (continues) vs. other sales where principal receives price (validated by istihsan, but not for sarf).
Practice applying the maxim to hypothetical cases.

Core Idea
An agent's ratification binds the principal when the principal is present, except in personal matters where the agent's utterance is essential.

In Hanafi agency law, an agent (wakil) can ratify acts done by an unauthorized person (faduli) or delegate authority without prior permission, and this binds the principal if the principal was present and aware. This rule rests on the rationale that the principal's presence fulfills the purpose of having his opinion represented. However, this rule has a critical exception: when the agency involves personal utterances like divorce or manumission, the agent's personal expression is the purpose, and ratification does not bind the principal. This lesson unpacks the qawida, its applications, and its exception.

The Principle of Mu'akhadha bi Z'amihi: Acknowledgment Binding Despite Contradictory Events
Further illustrations of iqrar remaining valid despite later contradictions

Learning Objective
By the end of this lesson, you will be able to apply the Hanafi maxim 'mu'akhadha bi z'amihi' to identify when an iqrar remains binding despite later contradictory events.

Core Idea
A person is bound by his own acknowledgment (iqrar) even if later events contradict it — the iqrar is not invalidated.

The Hanafi school places great weight on a person's own statements (iqrar). Even if later events seem to nullify the acknowledgment, the iqrar remains effective. The following cases from al-Ashbah wa al-Naza'ir illustrate this principle, showing that the initial declaration creates a legal obligation that does not dissolve merely because circumstances change.

Guided Path
1
Identify the iqrar and the later contradictory event
For each case, first identify what the person acknowledged (iqrar) and what later event appears to conflict with that acknowledgment. For example, in the first case: the buyer acknowledged the seller's ownership, but later the property was reclaimed by a third party.
The iqrar must be clear and explicit (sarih).
2
Determine whether the iqrar would be invalidated by standard rules
Consider whether the later event would normally render the iqrar void under normal fiqh principles. For instance, if the buyer had acknowledged ownership and then lost the property, one might think the acknowledgment is no longer relevant.
Standard iqrar rules: iqrar is binding against the person, but if the subject is impossible or the iqrar is contradicted by clear evidence, it may be void.
3
Apply the maxim 'mu'akhadha bi z'amihi'
The maxim states that a person is held to his own claim. Even if the later event contradicts the factual basis of the iqrar, the iqrar itself is not nullified. For example, the buyer's acknowledgment that the seller owns the property remains valid even after the property is reclaimed; so if the property returns to the buyer, he must deliver it to the seller.
The principle is based on the idea that a person's words are evidence against himself, and consistency is not required.
4
Identify the practical consequence (athar)
State what must happen as a result of the iqrar remaining binding. In the first case: the buyer must return the property to the seller. In the second: the earlier acknowledgment of the child's lineage (through the judge's order) stands despite later li'an. In the third: the slave becomes free even though the buyer later claimed ownership. In the fourth: the property becomes waqf even if later acquired.
The consequence is that the iqrar creates a legal obligation that cannot be undone by subsequent events.
5
Distinguish from cases where the iqrar would be invalid
Note that the maxim applies only when the iqrar was valid at the time it was made. If the iqrar itself was inherently impossible (e.g., claiming to own something that cannot be owned), it would be void from the start. The later event does not invalidate; it only highlights that the iqrar was already binding.
An iqrar that is false but plausible is binding; an iqrar that is impossible is void.

Iqrar vs Insha': The Power of Acknowledgment
Legal principle on the scope of iqrar versus insha'

Learning Objective
By the end of this lesson, you will be able to apply the qa'ida 'Yamlik al-iqrar ma la yamlik al-insha'' to distinguish situations where iqrar is effective and insha' is not, and construct a fatwa using this principle.

Core Idea
Iqrar bypasses capacity and consent requirements that block insha'.

This lesson introduces a foundational principle in Hanafi muamalat: the power of iqrar (acknowledgment) to produce legal effects where insha' (creating a new legal act) is invalid or ineffective. Through examples from joint debt, qadhf, and stipulated shares, you will learn how this principle operates and how to apply it in fatwa.

Guided Path
1
State the maxim
Write down the Arabic maxim and its meaning: by iqrar one can achieve what one cannot by insha'.
Memorize this as a key tool for fatwa.
2
Understand the rationale (talil)
Iqrar is an acknowledgment of a pre-existing right, so it does not create a new obligation requiring capacity or consent. Insha' creates a new legal effect and thus is subject to stricter rules.
This explains why iqrar can succeed where insha' fails.
3
Analyze the first furu'
Consider the joint debt example: one creditor cannot unilaterally defer his share via insha', but can acknowledge that the debt was originally deferred (iqrar).
Iqrar here changes the legal characterization without requiring the other creditor's consent.
4
Analyze the second furu'
In qadhf, the victim cannot pardon the defamer (insha'), but can acknowledge that his accusation was false (iqrar), which drops the hadd.
The victim's iqrar of falsity effectively nullifies the claim, unlike a pardon which is an insha' of forgiveness.
5
Analyze the branching furu'
If a person was stipulated a quarter share, his iqrar that someone else deserves it is valid; but making it for someone else via insha' is not.
Iqrar transfers the right by acknowledgment, while insha' (e.g., gift) would require capacity and delivery.

Testimony of a Wounded Person Regarding the Wound

Core Idea
A wounded person's denial of the wound is invalid if the wound is publicly known; otherwise, it is accepted on possibility of truth.

In Hanafi jurisprudence, the testimony of a wounded person denying his wound is not always accepted. The key factor is whether the wound is a known fact in the community. This ruling protects the right of retaliation (qisas) from being nullified by a possibly coerced or false denial. Understanding this rule helps a mufti navigate cases where the victim's statement conflicts with established reality.

The footnote clarifies details: if the victim was two sons (or slaves) and the acknowledger says he killed them, but the plaintiff says you killed two, the acknowledger's word is taken unless he named different names. So it is treated as one iqrar for one act, unless distinct victims are specified.

Leasing Trees for Fruit is Invalid
Ruling on leasing trees for fruit and similar cases

Learning Objective
By the end of this lesson, you will be able to state the Hanafi ruling on leasing trees for fruit and apply the qawida that the usufruct must be clearly defined and permissible.

Core Idea
Leasing a tree for its fruit is invalid because fruit is not a defined usufruct of the tree under ijarah.

This lesson examines a specific ijarah case: leasing trees or vines with the fruit going to the lessee. The Hanafi school deems this invalid because the fruit is not considered a manfa'ah (usufruct) of the tree itself. We will explore the underlying qawida and a minority opinion within the school.

Guided Path
1
Identify the Act and Its Ruling
The act is leasing a tree or vine with the condition that the fruit belongs to the lessee. The Hanafi ruling is that this lease is invalid (fasid).
2
State the Governing Qawida
The relevant legal maxim is that the usufruct (manfa'ah) in an ijarah contract must be clearly defined and permissible. The fruit of a tree is not a manfa'ah of the tree itself; rather, it is a separate asset. Thus, the lease lacks a valid subject matter.
This qawida is classified as a wusta (intermediate) maxim.
3
Examine the Dispute (Kharazadah's View)
Kharazadah mentions that some scholars argue that leasing a tree absolutely could be interpreted as referring to its usufruct (e.g., hanging clothes, tethering animals). They claim the lease could be valid if intended for such uses. However, the majority Hanafi position rejects this because the intended benefit is usually the fruit.
This is a legitimate ikhtilaf within the school.
4
Apply the Mufta Bihi Position
The relied-upon position for fatwa is that leasing trees for fruit is invalid. The mufti should advise the inquirer to structure the transaction as a sale of fruit or a sharecropping contract (muzara'ah) instead.
The fatwa hierarchy places the majority view over Kharazadah's isolated opinion.

Hiring Dirhams for Work is Invalid; Decoration is Valid
Ruling on hiring dirhams for work vs. decoration

Learning Objective
By the end of this lesson, you will be able to distinguish between valid and invalid hiring of dirhams based on the nature of the work and the requirement of a defined usufruct.

Core Idea
Hiring dirhams to work on them (e.g., melting) is invalid; hiring them for decoration (adding value) is valid if timed.

This lesson covers the ijarah of dirhams (silver coins). The Hanafi school rules that hiring dirhams for a manufacturing process that consumes them is invalid, but hiring them for decoration (tazyin) is permissible if the time is specified. The key is whether the benefit is a defined usufruct distinct from the substance.

Guided Path
1
Identify the Two Scenarios
First scenario: hiring dirhams to 'work on them' (e.g., melt, shape, or use as raw material) for a monthly fee. Second scenario: hiring dirhams to 'decorate them' (e.g., adding ornamental value without consuming the metal).
2
Apply the Ruling to the First Scenario
The first is invalid (fasid). Reason: The dirhams themselves are being consumed or transformed; the lessee is not returning the same dirhams. This is akin to a sale or a manufacturing contract (istisna), not ijarah. The lessee must return the dirhams and guarantee any loss.
The invalid contract is treated as a loan (qard) of the dirhams, hence the obligation to return and guarantee.
3
Apply the Ruling to the Second Scenario
The second (decoration) is valid if the time is specified. Reason: The dirhams remain intact, and the lessee benefits from their value while performing a permissible work (adding decoration). This is a valid ijarah of the dirhams' usufruct (their use as a medium for decoration).
The time condition is necessary to avoid gharar in the duration of the lease.
4
Identify the Governing Qawida
The qawida is that the subject of ijarah must be a defined usufruct that can be returned. In the first case, the usufruct (melting) consumes the substance, making it invalid. In the second, the usufruct (decorating) does not consume and is permissible.
This aligns with the maxim that 'the leased object must be returned' (al-ajānib yuridd).
