Using the Fiqh Index: Navigating al-Zarqa's Al-Madkhal al-Fiqhi al-'Amm
Alphabetical index of fiqh topics with page references

Learning Objective
By the end of this lesson, you will be able to use the alphabetical index to locate specific Hanafi usul and fiqh topics, interpret page references, and identify cross-references.

Core Idea
A well-structured index is a map to the madhhab's legal reasoning.

This passage is an alphabetical index of topics from Mustafa Ahmad al-Zarqa's 'Al-Madkhal al-Fiqhi al-'Amm', a comprehensive introduction to Islamic jurisprudence. While not a ruling or proof, the index is a vital tool for navigating the book's rich discussions on Hanafi usul and fiqh. Understanding its structure will help you efficiently access the scholarly debates, definitions, and applications you need.

Guided Path
1
Identify the index structure
Notice that entries are ordered alphabetically by Arabic root. Each main entry (e.g., الإباحة) is bolded, followed by sub-entries and page references.
The index uses a simple alphabetical ordering, not by letter but by root word.
2
Interpret the page references
Page references like 4/23 indicate volume/page. The first number is the volume, the second is the page. For example, '٤/٢٣' means volume 4, page 23.
Some references may have line numbers or multiple pages separated by commas.
3
Recognize cross-references
Entries like 'انظر: أهلية الجنين' direct you to another main entry. Use these to find related discussions.
Cross-references help connect topics that are discussed under different main headings.
4
Locate a specific usul topic
Find 'الاستحسان' and note its page references and sub-entries. This will help you quickly access discussions on istihsan in the text.
The entry for 'الاستحسان' includes references to multiple volumes and pages, indicating its extensive treatment.
5
Use the index for Hanafi study
Understand that this index covers not only rulings but also usul principles like ijtihad, ijma, and istihsan. Use it to build a systematic review of key concepts.
The index is especially useful for students of Hanafi usul al-fiqh and iftaa.

Index listing fiqh topics with page references

Learning Objective
By the end of this lesson, you will be able to identify three major Hanafi legal maxims listed in the index and state their page references.

Core Idea
An index is a scholar's compass: it shows where to find rulings and maxims.

This passage is an alphabetical index from al-Madkhal al-Fiqhi al-'Amm by Mustafa al-Zarqa. It lists numerous fiqh topics and includes references to foundational Hanafi maxims like 'المشقة تجلب التيسير' and 'الأمور بمقاصدها'. While an index does not contain rulings itself, it is an essential tool for any mufti or researcher to locate discussions quickly. In this lesson, we will practice reading the index and understanding its structure.

Guided Path
1
Locate the legal maxims
Scan the Arabic entries for phrases that are clearly maxims, usually under separate headings like 'المشقة', 'الأمور', 'المصالح' etc.
These are not rulings but principles that guide rulings.
2
Note the page references
Each maxim is followed by a page number like 30/81. The first number is the volume, the second is the page.
Al-Madkhal has multiple volumes.
3
Understand cross-references
Some entries say 'انظر' (see) linking to related topics. For example, 'المصالح المرسلة' has a cross-reference to 'الاستصلاح'.
This helps navigate the book.
4
Identify the rank of the maxim
From the key terms, we learn that 'المشقة تجلب التيسير' is one of the five major maxims (al-qawa'id al-kubra).
Rank determines its scope and precedence.
5
Apply the index in fatwa preparation
When faced with a question about concessions, a mufti would check the index for 'المشقة', then turn to the referenced page for detailed discussion.
This saves time and ensures thoroughness.

Learning Objective
By the end of this lesson, you will be able to identify five major Hanafi legal maxims from an index, explain their meanings, and apply them to fatwa scenarios.

Core Idea
Five Hanafi legal maxims from the index form the backbone of many fatwa rulings.

This passage is an alphabetical index, but it contains the full text of several foundational Hanafi legal maxims. These maxims are not just organizational tools; they are operative principles that judges and muftis use to resolve conflicts and derive rulings. In this lesson, we will extract and examine five key maxims that appear on this page, building your ability to recognize and apply them.

Guided Path

Locate the maxims in the index
Scan the index entries for phrases that look like legal maxims (qawa'id). They appear under entries like 'المفسدة', 'المقصد', 'الملكية', etc. Write down the exact Arabic phrases.

3
