The Proof That Relations Have No External Existence

Learning Objective

Core Idea
If a relation existed externally, its inherence would create an infinite regress of relations.

The refutation maintains that the original argument for indivisibility does not rely on the external existence of contact; rather, it follows from the necessity of two distinct directions in a single atom, which requires division in the atom itself, not in relations.

This passage is a dialectical argument defending the atomist position (the existence of the indivisible atom) by showing that the alternative views lead to greater absurdities. The author begins by recounting a previous argument about infinite points in a circle, which is shown to be fallacious. Each impossibility is claimed to be more severe than accepting the disintegration of the millstone. The argument uses a ranking of absurdities to show that the atomist position is the least absurd, hence must be accepted. The passage is cut off at the fourth impossibility, which is incomplete.

Commentary
This passage is an objection-and-answer argument. The objector claims that if an atom moves from one part to another, it cannot be moving while on the first part (since motion hasn't started) nor after reaching the second (since motion has ended), so motion must occur in an intermediate state between the two positions. The author, al-Razi, refutes this by drawing an analogy with the transition from non-existence to existence: if we accept the objector's reasoning, we would have to affirm an intermediate state between non-existence and existence, which is absurd. The only way to avoid this absurdity is to say that the transition occurs instantaneously with no intermediate state. The author then applies the same logic to the atom: the atom goes directly from contact with the first part to contact with the second part, with no real intermediate state; the imagined intermediate is merely a product of illusion. To further prove this, he argues that if there were a real intermediate state of motion, the atom would have to be in a specific location (contradicting motion), in a non-specific location (impossible for a determinate existent), or in no location (impossible for a volume). Therefore, motion is nothing but the first obtaining in the second location, and consecutive such obtainings constitute motion. Thus the objection is invalid.

Refuting the Argument from Shadow to Body Divisibility
Razi refutes an argument linking shadow length to body divisibility.

Learning Objective
By the end of this lesson, you will understand how Razi dismantles the objection that a body's shadow being twice its length proves the body is divisible.

Core Idea
An inference from shadow length to body divisibility fails if the body has no half.

The fifth argument claims that if a body's shadow is twice its length, then half the shadow must correspond to half the body, implying divisibility. Razi's response is subtle: he accepts the premise but denies the inference where the body lacks a half. Let's examine this logical move closely.

Guided Path
1
Read the objection carefully
Read the objector's argument: the body has a shadow twice its length, so half that shadow equals the shadow of half the body, hence the body has a half and is divisible.
Note the logical structure: premise → intermediate step → conclusion.
2
Identify Razi's refutation
Razi distinguishes two cases: if the body has a half, the inference holds; if not, we simply say the shadow is twice the body without attributing a half.
The key is that the argument assumes the body has a half—the very point in dispute.
3
Reflect on the circularity
The objector's reasoning is circular: it uses the existence of a half to prove divisibility, but the existence of a half already implies divisibility.
This is a classic refutation that exposes a begged question.

Refuting the Shadow Argument for Atomism
Refutation of the fifth argument for atomism from shadows.

Learning Objective
By the end of this lesson, you will understand how the shadow argument for atomism is refuted by distinguishing between a body that has a half and one that does not.

Core Idea
The shadow argument only works if atoms already have halves.

The fifth argument for atomism tries to use everyday observations about shadows to prove that bodies must be divisible. The argument sounds plausible, but a careful logical response shows it assumes what it tries to prove. In this lesson, we will examine the argument and its refutation, focusing on the key distinction that undermines the conclusion.

The argument says: a body can have a shadow twice its length (e.g., at a certain time of year). If that happens, then half of that shadow would equal the body itself. Therefore, the body has a half and is divisible.

The Logical Dependence Argument against Atomism
First logical argument against atomism from dependence of knowledge on constituents.

Learning Objective
By the end of this lesson, you will understand how the first logical argument against atomism uses the principle that knowledge of a composite quiddity depends on knowledge of its constituents to disprove indivisible parts.

This is a powerful reductio based on common awareness.

The objection is sharp, but the author disarms it by turning it against the opponent's own metaphysical commitments. The objection examines both possibilities: if parts acquire length and width when joined, then each part becomes long and wide, which regenerates the problem for the parts. If not, then the whole has no magnitude at all, which is false.
