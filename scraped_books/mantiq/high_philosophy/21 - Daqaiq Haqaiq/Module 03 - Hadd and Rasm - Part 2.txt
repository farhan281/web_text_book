Core Idea
Start with the genus, then the differentia — order matters in definition.

In defining a thing, the usual order is to state the genus first, then the differentia. But what if we reverse them? This passage from the classical logic text explores why starting with the differentia is disliked, presenting three arguments and dismissing one as insubstantial. Let's examine the reasoning.

The Second Definition: Essential Distinction without Completeness

Learning Objective

Core Idea
The second definition gives essential distinction through a single differentia, even if many essential attributes exist.

The complete real definition (al-hadd al-tam) aims to list all essential attributes, but the second definition (al-hadd al-thani) has a more modest goal: it only needs to provide an essential conception that distinguishes the thing. However, when a thing has multiple inseparable differentiae, one might mistakenly think that all are needed. This lesson will help you understand the nature and utility of the second definition.

Sufficiency of One Differentia in Definition
Author discusses whether all differentiae must be mentioned when multiple exist.

Learning Objective
By the end of this lesson, you will be able to evaluate whether a definition must include all concurrent differentiae and explain why only one essential differentia is required.

Core Idea
One essential differentia suffices for definition because it provides essential distinction, not absolute distinction.

When a species has multiple differentiae that are all inseparable from it, a question arises: must a definition mention all of them? The author refutes this view by distinguishing between essential and accidental concomitance. This lesson unpacks that distinction and shows why one differentia is enough.

Guided Path
1
Identify the objection
Notice the claim: if multiple differentiae are equally concomitant to a species, then each could be used to distinguish it; but using one would equally distinguish the other, so all must be mentioned.
The objection assumes that distinction must be absolute, not essential.
2
Understand the author's distinction
The author introduces 'essential distinction' (al-tamyiz al-dhati) — the differentia is meant to distinguish the species by its constitutive, prior meaning, not by any accidental concomitant.
Essential distinction is unique to the differentia; accidental concomitance does not provide that.
3
Analyze the refutation
The author argues that the concomitance among differentiae is accidental, not essential, while each differentia is essentially concomitant to the species. Therefore, the intended distinction is not shared.
This is why one differentia alone suffices for a complete definition.
4
Apply to a real definition
Think of a definition like 'Human is a rational animal'. 'Rational' is the essential differentia. Even if 'laughing' is also concomitant, it is not needed in the definition because it is not prior or constitutive.
This shows the practical implication of the rule.

Evaluate the overall refutation
The author concludes that the opponent's argument is necessary for him (i.e., leads to his own position) when the constitutents differ by genus and differentia, implying no escape from the conclusion.

More hidden cannot make the more apparent known, so it fails. Equal gives no priority—defining one by the other is arbitrary. Only 'more known' remains valid.

Guided Path

Understand the two conditions
Iṭṭirād means that whenever the definition exists, the defined exists. In'ikās means that whenever the definition is absent, the defined is absent. Together, they guarantee co-extensiveness.

The Three Checks for a Valid Definition
Author establishes the primary checks for definition validity.

Learning Objective
By the end of this lesson, you will be able to identify whether a definition is true (sādiq), coextensive (muṭṭarid), and reciprocal (mun‘akis) with its definiendum.

Core Idea
A definition must be true, cover all instances, and exclude everything else.

When evaluating a definition, the very first step is to check its relationship to the definiendum. Is it true? Does it apply to every individual? Does it apply only to those individuals? This lesson walks you through these three essential criteria using examples from classical logic texts. Master these checks to quickly spot many invalid definitions.

Guided Path
1
Check truth (ṣidq)
Ask: Is the definition actually true of the thing being defined? If not, the definition is immediately invalid. For example, defining 'human' as 'neighing animal' is false because humans do not neigh.
2
Check coextension (iṭrād)
Ask: Does the definition cover every instance of the definiendum? If some instances are left out, the definition is too narrow. For example, defining 'human' as 'rational animal' is coextensive if all humans are rational animals and no rational animal is non-human.
3
Check reciprocity (in‘ikās)
Ask: Does the definition apply only to the definiendum? If it also applies to things outside, it is too broad. For example, defining 'animal' as 'living thing' is not reciprocal because plants are also living.
A valid definition must satisfy all three: truth, coextension, and reciprocity.
4
Apply to an example
Take the definition of 'substance' as 'that which exists in a subject'. It fails the truth test because that description actually fits accidents, not substances. Hence it is invalid without further checks.

Truth and Coextensiveness Checks for Definitions
Author introduces the two primary checks for whether a definition is defective: truth and coextensiveness.

Learning Objective
By the end of this lesson, you will be able to identify whether a definition is defective based on the checks of truth (sidq) and coextensiveness (itirāḍ).

Core Idea
A true definition must be both true of and coextensive with the definiendum.

When encountering a definition, the first thing to verify is whether it is true of the thing being defined. If it is false, like calling man 'the neighing animal', the definition is immediately invalid. If it is true, you must then check whether it is coextensive – that is, it applies to every instance of the definiendum. Only after passing both checks can you look for other potential defects.

Guided Path
1
Identify the definiendum and definition
Find the term being defined (al-mahdūd) and the proposed definition (al-ḥadd). In the passage, examples are given: substance (jawhar) defined as 'that which exists in a subject', and man (insān) defined as 'the neighing animal'.
2
Apply the truth check (ṣidq)
Ask: Is the definition true of every individual of the definiendum? For the first example, 'that which exists in a subject' is actually the definition of accident, not substance – so it is false. For the second, 'neighing' is not true of all humans, so it is false.
If false, the definition is defective; no further checks needed.
3
Apply the coextensiveness check (iṭṭirād)
If the definition is true, ask: Does it apply to all instances of the definiendum (i.e., is it coextensive)? For example, defining man as 'animal' is true but not coextensive, because it includes non-human animals. Such a definition is defective.
Coextensiveness means the definition covers exactly the set of things that are the definiendum.
4
Combine both checks
Only definitions that are both true and coextensive pass the initial screening. The author then proceeds to examine other potential defects for such definitions.
These two checks are preliminary; other defects like obscurity or circularity may remain.

Three Meaning-Related Defects in Definitions
Author explains defects in definition related to meaning: sameness, obscurity, and circularity.

Learning Objective
By the end of this lesson, you will be able to identify and classify three defects in definitions related to meaning: identity, obscurity, and circularity.

Core Idea
A good definition must be clearer than the definiendum; otherwise it suffers from tautology, obscurity, or circularity.

A definition is meant to clarify a term. But if the definer is not clearer, the definition fails. In this lesson, we explore three types of meaning-related defects: when the definer is identical to the definiendum (tautology), when it is more obscure, or when it is equally obscure (leading to circularity or arbitrariness). We will learn to spot these flaws in definitions across the Islamic sciences.

Guided Path
1
Check for identity
Examine whether the definer is exactly the same as the definiendum. For example, defining 'man' as 'human' is a tautology. Note that in verbal definitions (al-lafẓī), identity is acceptable because you are explaining a word's meaning, not the essence.
Tautology is the simplest defect to detect.
2
Check for obscurity
If the definer is different, ask: Is the definer more obscure than the definiendum? If yes, the definition is defective because it fails to clarify. The author gives two subtypes: dependence and non-dependence.
More obscure means the definer is less known than the term being defined.
3
Detect dependence in obscurity
If the definer is more obscure, see whether understanding the definer depends on understanding the definiendum. Example: defining the sun as 'a planet that rises by day' – day depends on the sun, so the definition is circular (dawr). This is the worst form.
Dependence creates a vicious circle.
4
Detect equal obscurity
If the definer is not more obscure, check if it is equally obscure. Then check whether the definer and definiendum are mutually dependent (like father and son) – this leads to circularity. If not mutually dependent (like black and white), the definition is arbitrary, with no reason to choose one over the other.
Equal obscurity without mutual dependence is still defective because no priority.
5
Apply the rule
The author's principle: a definition must be by something more known than the definiendum, not by something known with it. Test any definition in Islamic sciences against this rule.
This rule prevents all three defects.

Two Types of Genus Defects: Omission and False Genus
Two genus defects: omission and false genus with examples

Learning Objective
By the end of this lesson, you will be able to identify whether a complete definition fails due to missing genus or due to using a term that is not a true genus, and provide a correction.

Core Idea
A complete definition demands a true proximate genus; omission or a false genus breaks it.

After understanding that genus defects only occur in complete definitions, the author explains two specific ways a genus can be defective: it may be omitted altogether, or what is taken as the genus may not actually be a genus. He illustrates with examples from psychic dispositions and physical movements. This lesson sharpens your ability to spot these errors in Islamic scientific definitions.

Guided Path
1
Find the genus in the definition
Locate the term that is supposed to be the genus. In a complete definition, the genus is the broader category. For example, in 'man is a rational animal', 'animal' is the genus.
If no genus term exists, the definition is incomplete (omission defect).
2
Check if the genus is present
If the definition lacks a genus entirely, it is a case of omission. The author gives the example of defining man using only 'rational' (differentia) or 'laughing' (property) without 'animal'.
Omission defect: the definition is incomplete because the genus is missing.
3
Test if the supposed genus is truly a genus
The supposed genus must be true of all individuals of the species. The author says to check if it is 'صادقاً في الحمل على المفروض نوعاً' (true in predication upon the supposed species). If not, it is a false genus.
For example, 'disposition' is not true of 'action' because not every action is a disposition.
4
Identify the category mismatch
The false genus occurs when the supposed genus belongs to a different category than the species. The author gives examples: taking 'disposition' (a quality) as genus for 'action' (an action), or 'action' as genus for 'disposition'. Also, taking 'passive' as genus for 'passion'.
The error is a cross-category predication.
5
Propose a correct genus
Once the false genus is identified, suggest a proper genus that truly includes the species. For 'recollection', a correct genus might be 'mental act' rather than 'disposition'. For 'wind', the genus could be 'movement' rather than 'air'.
The correct genus must be predicable of the species and be a genuine category.

Circular Definitions: When the Definiens Is Not More Known
Author identifies circularity as a defect when definiens and definiendum are equally known.

Learning Objective
By the end of this lesson, you will be able to identify circular definitions and distinguish between mutual dependence and equal-knowledge cases.

Core Idea
A valid definition must define the unknown by the known, not the equally known.

In defining terms, one rule is cardinal: the definiens must be better known than the definiendum. The author here warns against a subtle defect where the definiens is not more known but equally known, leading to circular definitions. We will examine two types: mutual dependence (father-son) and equal knowledge (black-white).

Guided Path
1
Identify the term being defined
Find the definiendum in the example. For instance, in 'father is one who has a son', the term 'father' is being defined.
2
Check if the definiens is more known
Compare the knowledge of the definiens and definiendum. In the father-son case, 'son' is as known as 'father' – neither is prior.
The author stresses that definitions should proceed from the known to the unknown.
3
Detect mutual dependence or equal knowledge
Determine if the definiens depends on the definiendum (as in father-son) or if they are independent but equally known (as in black-white). Both fail the rule.
In the first case, circularity is explicit; in the second, no priority justifies one being the definiens.
4
Apply the rule
Conclude whether the definition is valid or corrupt. If the definiens is not more known, the definition is invalid due to circularity or equal knowledge.
This rule prevents hidden defects in definitions.

Genus Defects: Omission and False Genus
Author explains omission and false genus with multiple tests.

Learning Objective
By the end of this lesson, you will be able to identify whether a genus defect is due to omission or a false genus, and apply the five indicators of a false genus to evaluate a definition.

Core Idea
A false genus fails to be truly predicable and essential to all species members.

Building on the categorization of defects, the author now delves into the two main genus defects: omitting the genus entirely, and using a term that is not a true genus. He supplies several tests to detect a false genus, including predication failure, partial coverage, differentia mismatch, and more. These tests are crucial for evaluating definitions in any Islamic science.

Guided Path
1
Distinguish omission from false genus
The author says: 'فإما لعدم ذكره أو لعدم جنسية ما فرض جنساً'. First, check if the genus is completely absent (e.g., defining man only by 'rational'). If present but wrong, proceed to false genus tests.
Omission is the easiest to spot; false genus requires deeper analysis.
2
Apply the predication test
The first test: 'أن لا يكون المفروض جنساً صادقاً في الحمل على المفروض نوعاً'. Check if the assumed genus is truly predicated of every individual of the species. If not, it is a false genus.
Example: 'recollection is a psychological disposition' – 'disposition' is not predicated of 'act'.
3
Apply the partial coverage test
The second test: even if true of some species members, it may fail for others that differ only accidentally, like 'indivisible' for lines (true for non-straight lines but false for divisible ones).
The genus must cover all species members; accidental differences should not exclude.
4
Apply the differentia and property test
The third test: 'أن يكون المفروض جنساً فصوله غير صادقة على ما فرض نوعاً'. If the genus's differentiae or properties do not apply to the species, the genus is false.
Since the differentia of the genus is essential to the genus, it must also be essential to the species.
5
Apply the remaining tests
The fourth test: a predicate true of the species but not of the genus. The fifth: genus and species exist in different subjects (e.g., anger in irascible, pain in sensation). The sixth: the assumed genus is actually a species of the assumed species. The seventh: even if broader, the term may be equivocal or accidental.
Each test provides a clear criterion to reject a false genus.

The Accidental Genus Fallacy

Learning Objective
By the end of this lesson, you will be able to identify when a proposed genus is invalid because it applies to some species only accidentally.

Core Idea
A valid genus must be essential for all its species; accidental applicability is not enough.

In defining logical terms, the genus must be truly essential. This passage highlights a specific fallacy: proposing a genus that is true of some species but not others, where the difference is merely accidental. Understanding this helps avoid flawed definitions in Ilm al-Mantiq and beyond.

Identify the proposed genus and species

If a predicate is not essential for one of two co-extensive things, it is not essential for the other. Since 'indivisible' is not essential for divisible non-straight lines, it cannot be essential for non-straight lines in general.

Why a Genus's Differentia Must Apply to Its Species
Explains fallacy when genus's differentiae do not apply to the supposed species

Learning Objective
By the end of this lesson, you will be able to identify when a proposed genus is invalid because its differentiae do not apply to the supposed species.

Core Idea
The differentia of a genus must be true of every species under it — essentiality transfers.

This passage from Daqaiq al-Haqaiq highlights a critical test for correct genus-species relationships. When checking a definition, you must verify that the differentiae of the genus actually hold for the supposed species. If they don't, the genus is false, because what is essential to the genus is essential to the species. The same applies to properties (khassa) of the genus.

Guided Path
1
Identify the genus and species
In any definition, locate the proposed genus (جنس) and the supposed species (نوع). The genus is the broader category, the species the narrower.
For example, in defining 'human' as 'rational animal', 'animal' is the genus, 'human' the species.
2
List the differentiae of the genus
Write down the differentiae (فصول) that the genus is said to have. These are essential attributes that distinguish it from other genera.
Differentiae are not just any attributes — they are essential to the genus itself.
3
Check each differentia against the species
Test whether each differentia of the genus applies to the supposed species. If even one differentia is false of the species, the genus is invalid.
The text states: 'the essential of the essential is essential' — so what is essential to the genus cannot be absent from the species.
4
Apply the same test to properties (khassa)
The same rule holds for properties (خاصة) of the genus. If a property of the genus does not hold for the species, the genus is also invalid.
Properties are non-essential but co-extensive with the genus; they too must extend to the species.
5
Understand the logical principle
Why must this hold? Because the differentia is essential (ذاتي) to the genus. Since the genus is essential to the species, what is essential to the genus is also essential to the species. This transitivity ensures consistency.
This principle is known as 'ذاتي الذاتي ذاتي' (the essential of the essential is essential).

Distinguishing Subject from Genus in Accidental Definitions
Author states that the subject cannot be used as the genus for an accident.

Learning Objective
By the end of this lesson, you will be able to identify when a definition mistakenly uses the subject of an accident as its genus, and correct it.

Core Idea
The subject enables existence; the genus defines essence.

When defining an accident, it is tempting to use the thing it inheres in as its genus. But a genus must be constitutive of the essence, not merely necessary for existence. This lesson dissects that error through concrete examples from the tradition.

Guided Path
1
Identify the accident and its subject
From the definition, extract what is being defined (the accident) and what it is said to be 'of' (the subject). For example, in 'immortality is eternal life', the accident is immortality, the subject is eternal life.
The subject is often introduced with the preposition 'of' or appears as the second term in a genitive construct.
2
Check whether the subject is essential to the accident's concept
Ask: Is the subject part of the meaning of the accident, or is it merely something the accident happens to? For immortality, the concept does not include 'eternal life' intrinsically; it is the state of not dying that can occur in eternal life.
If the accident can be imagined without that specific subject, the subject is not a genus.
3
Test whether the subject is preserved in the accident's existence
For 'corrupt' subjects (like wood in ash), the subject ceases to exist when the accident occurs. A genus must persist in the species. For 'existent' subjects (like eternal life in immortality), the subject remains but is still not part of the essence.
The author distinguishes between constitution in existence vs. constitution in reality (essence).
4
Replace the mistaken genus with a proper one or label it 'subject'
If the term given as genus is actually a subject, note that it should be classified as a subject (mawdu'), not a genus. A correct definition would need a different, more abstract genus that captures the accident's essence.
For example, 'immortality' might have 'state' as a genus, and 'frozen water' is better defined as 'water that has congealed' with 'water' as subject.
